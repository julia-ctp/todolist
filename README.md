# To Do List

A simple and modern application for task management. 

---

## How to run the project

```bash
# Install dependencies 
npm install

# Run the server 
npm run dev
```

## Technologies

* **Next.js 16**
* **TypeScript**
* **Shadcn** 
* **TailwindCSS**
* **React Hook Form**
* **Zod** 
* **Axios**
* **Biome** 
* **Prettier**
